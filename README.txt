INTRODUCTION
------------
This project integrates PayWay NET online payments into
the Drupal Commerce payment and checkout systems.
https://www.payway.com.au/docs/rest.html
* To submit bug reports and feature suggestions, or to track changes:
  https://www.drupal.org/project/issues/commerce_payway_net

REQUIREMENTS
------------
This module requires the following:
* Submodules of Drupal Commerce package (https://drupal.org/project/commerce)
  - Commerce core,
  - Commerce Payment (and its dependencies);
* PayWay account with Net setup (https://www.payway.com.au/net/NetSetupView).
